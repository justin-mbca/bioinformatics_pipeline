# Pipelines

This folder contains templates and scaffolding for pipelines used in this project.

- `genomics/` — Nextflow and Snakemake templates for sequencing analysis (QC, alignment, variant calling, annotation)
- `clinical/` — Prefect-based skeleton for clinical data ingestion, cleaning, and statistical analysis
- `integration/` — Examples showing how to connect genomics outputs to clinical analyses

See each subfolder for usage notes and to customize for your data and compute environment.
